# BMI Calculator React App

A responsive and beginner-friendly **BMI (Body Mass Index) Calculator** built using **React**. The app takes **weight in kilograms** and **height in feet**, calculates the BMI using standard formulas, and displays the BMI value along with the corresponding weight category.

---

## Features

* Accepts height in **feet** and weight in **kg**
* Converts height to meters internally for accurate BMI calculation
* Real-time feedback with weight categories:

  * Underweight
  * Normal
  * Overweight
  * Obese
* Fully responsive layout
* Clean and beginner-friendly code structure
